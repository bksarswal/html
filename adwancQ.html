Advanced Questions

1. What are custom data attributes in HTML?

 
    Custom attributes start with data- and store extra data on HTML elements.
    html
    Copy code
    <.div data-user-id="12345"><./div>
    Access using JavaScript:
    javascript
    Copy code
    const userId = document.querySelector('div').dataset.userId;
    console.log(userId); // Output: 12345

2. What is the difference between <.script> and <.script async>?

 <.script>: The browser loads and executes the script immediately, blocking rendering.
    <.script async>: The script loads and executes asynchronously, without blocking rendering.

3. What is the purpose of the <.link> tag in HTML?

The <.link> tag connects external resources to the HTML document, such as stylesheets or icons. Example:
        html
        Copy code
         <.link rel="stylesheet" href="styles.css">

4. What is ARIA in HTML?

     ARIA (Accessible Rich Internet Applications) attributes improve accessibility for people using assistive technologies. Example:
    html
    Copy code
    <.button aria-label="Close menu">X<./button>

5. How can you improve the performance of an HTML page?

 Minimize and compress CSS, JavaScript, and HTML files.
    Use lazy loading for images and iframes (loading="lazy").
    Use content delivery networks (CDNs) for external resources.
    Optimize images using modern formats like WebP.
