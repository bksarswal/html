Basic Questions

1. What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design the structure of web pages. It uses tags to define elements like headings, paragraphs, links, images, etc.

2. What are the different types of HTML tags?

Structural tags: <.html>, <.head>, <.body>, <.header>, <.footer>, <.main>, <.section>, etc.
Formatting tags: <.b>, <.i>, <.u>,<.s> for like This <.strong>, <.em>, <.mark>.
Link and media tags: <.a>, <.img>, <.audio>, <.video>.
Table tags: <.table>, <.tr>, <.td>, <.th>, <.caption>.

3. What is the difference between <.div> and <.span>? 

<.div>: A block-level container used to group elements. It occupies the full width available.
<.span>: An inline container used to apply styles or group small parts of text within elements.

4. What is the purpose of the DOCTYPE declaration?

The declaration specifies the HTML version to the browser, ensuring the web page is rendered correctly. Example: for HTML5.

5. What is semantic HTML? Why is it important?

HTML uses tags that have meaningful names, such as <.header>, <.article>, <.nav>, <.footer>, etc.
Importance:
Improves accessibility for screen readers.
Helps search engines understand the structure of the webpage (SEO).
Makes code more readable and maintainable.
