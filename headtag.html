Head tag

 
    !The <.head> tag in HTML is used to contain metadata and 
    resources for  the document.
    Metadata is information about the document that is not
    displayed on the webpage but is essential for browsers,
    search engines,and other services.                                                                                                                                                                                  
   
 Syntax:

    html
 code
    <.head>
        
    <./head>

   
 Common Elements Inside <.head>:


   
 Document Title (<.title>):

    
    Specifies the title of the webpage, displayed in the browser's
     title bar or tab.
    html
 code
    <.title>My Webpage<./title>

  
  Character Encoding (<.meta charset>):

    
    Defines the character encoding of the document,
     ensuring proper display of text.
    html
 code
    <.meta charset="UTF-8">

   
 Viewport Settings (<.meta name="viewport">):
    
    Optimizes the webpage for mobile devices.
    html
    code
    <.meta name="viewport" content="width=device-width, initial-scale=1.0">
      

    
Description and Keywords (<.meta>):

    Provides a description of the webpage and keywords for search engines.
    html
    Copy code
    <.meta name="description" content="This is an example webpage.">
    <.meta name="keywords" content="HTML, head tag, example">

    
Stylesheets (<.link>):

    
    Links to external CSS files.
    html
    Copy code 
    <.link rel="stylesheet" href="styles.css">
     
   
Favicon (<.link rel="icon">):

    
    Specifies the website's favicon.
    html
    Copy code
    <.link rel="icon" href="favicon.ico" type="image/x-icon">
  

.Scripts (<.script>):

    
    Includes external or inline JavaScript.
    html
    Copy code
    <.script src=".script.js" defer>
   
    
Preloading Resources (<.link rel="preload">):

    
    Specifies resources to preload for better performance.
    html
    Copy code
    <.link rel="preload" href="font.woff2" as="font" type="font/woff2" crossorigin="anonymous">


 Other Metadata:

    
    
Open Graph tags for social media:

    html
    Copy code
    <.meta property="og:title" content="My Webpage">
    <.meta property="og:de.scription" content="An example webpage.">
  
  Robots for SEO:

    html
    Copy code
    <.meta name="robots" content="index, follow">
  
         
  Example of a <.head> Section:

    html
    Copy code
    <.head>
        <.meta charset="UTF-8">
        <.meta name="viewport" content="width=device-width, initial-scale=1.0">
        <.meta name="de.scription" content="Learn about the HTML head tag.">
        <.meta name="keywords" content="HTML, head, metadata">
        <.title>HTML Head Tag<./title>
        <.link rel="stylesheet" href="styles.css">
        <.link rel="icon" href="favicon.ico" type="image/x-icon">
        <.script src=".script.js" defer>
    <./head>
   
 Importance of the <.head> Tag:

    SEO: Metadata improves search engine visibility.
    Performance: Preloading and linking resources enhances performance.
    User Experience: Titles and favicons improve usability.
    Compatibility: Ensures proper rendering and behavior across devices and browsers.
