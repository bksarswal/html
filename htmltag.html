html tag


The <.html> tag in HTML is the root element of an HTML document.
     It encapsulates all other HTML elements and serves as
      the top-level container for the content and structure of the webpage.
       Every HTML document begins with the <.html> tag.

Syntax:
html
Copy code
<.!DOCTYPE html>
<.html lang="en">
<.head>
    <.title>Document Title

<.body>
    <.h1>Welcome to My Website
    <.p>This is a paragraph.




Key Features of the <.html> Tag:

Document Root:

The <.html> element is the root element of the document.
It contains two main child elements:
<.head>: Contains metadata about the document (e.g., title, character set, styles, scripts).
<.body>: Contains the visible content of the webpage.


Language Attribute (lang):


Specifies the language of the document for accessibility and SEO purposes.
html
Copy code
<.html lang="en">
Global Attributes:

Supports global attributes like class, id, style, and data-*, though they are rarely used with the  tag.
Doctype Declaration:

The <.!DOCTYPE html> declaration precedes the <.html> tag to specify that the document is an HTML5 document.
Example of a Minimal HTML Document:
html
Copy code
<.!DOCTYPE html>
<.html lang="en">
<.head>
    <.meta charset="UTF-8">
    <.meta name="viewport" content="width=device-width, initial-scale=1.0">
    <.title>Minimal HTML

<.body>
    <.h1>Hello, World!
<./body>
<./html>


Importance:

The <.html> tag ensures browsers and tools interpret the document as HTML.
It provides a structured foundation for web content.
Specifying attributes like lang supports internationalization and improves accessibility.
